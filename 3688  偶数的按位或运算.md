#### 3688  偶数的按位或运算
---


*01  随便写写*
```python
class Solution(object):
    def evenNumberBitwiseORs(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        result = 0
        for digit in nums:
            if digit % 2 == 0:
                result = result|digit
        return result
```
