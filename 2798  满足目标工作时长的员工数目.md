#### 2798  满足目标工作时长的员工数目
---


*01  随便写写*
```python
class Solution(object):
    def numberOfEmployeesWhoMetTarget(self, hours, target):
        """
        :type hours: List[int]
        :type target: int
        :rtype: int
        """
        cnt = 0
        for staff in hours:
            if staff >= target:
                cnt += 1
        return cnt
```
