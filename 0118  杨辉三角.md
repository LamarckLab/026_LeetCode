*01 自底向上构造 动态规划 o(n^2)*
```python
class Solution(object):
    def generate(self, numRows):
        """
        :type numRows: int
        :rtype: List[List[int]]
        """
        ans = [[1],[1,1]]
        if numRows <= len(ans):
            return ans[:numRows]
        for i in range(2,numRows):
            subLine = [1]*(i+1)
            for j in range(1,i):
               subLine[j] = ans[i-1][j-1] + ans[i-1][j]
            ans.append(subLine)
        return ans
```
