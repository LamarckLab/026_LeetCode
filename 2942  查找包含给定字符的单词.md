#### 2942  查找包含给定字符的单词
---


*01  随便写写*
```python
class Solution(object):
    def findWordsContaining(self, words, x):
        """
        :type words: List[str]
        :type x: str
        :rtype: List[int]
        """
        index_list = []
        for index in range(len(words)):
            if x in words[index]:
                index_list.append(index)
        return index_list
```
