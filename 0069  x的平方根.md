#### 0069  x的平方根
---


*01 二分查找o(logN)*
```python
class Solution(object):
    def mySqrt(self, x):
        """
        :type x: int
        :rtype: int
        """
        left, right = 0, x
        while left <= right:
            mid = (left+right) // 2
            if mid*mid < x:
                left = mid + 1
            elif mid*mid > x:
                right = mid - 1
            else:
                return mid
        return right
```
