#### 3285  找到稳定山的下标
---


*01  随便写写*
```python
class Solution(object):
    def stableMountains(self, height, threshold):
        """
        :type height: List[int]
        :type threshold: int
        :rtype: List[int]
        """
        stable_index_list = []
        for index in range(1,len(height)):
            if height[index-1] > threshold:
                stable_index_list.append(index)
        return stable_index_list
```
