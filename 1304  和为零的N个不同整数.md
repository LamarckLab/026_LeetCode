#### 1304  和为零的N个不同整数
---


*01  随便写写*
```python
class Solution(object):
    def sumZero(self, n):
        """
        :type n: int
        :rtype: List[int]
        """
        ans = []
        if n % 2 == 0:
            for i in range(1,n//2+1):
                ans.extend([i,-i])
        else:
            ans.append(0)
            for i in range(1,n//2+1):
                ans.extend([i,-i])
        return ans
```
