#### 1769  移动所有球到每个盒子所需的最小操作数
---


*01  两个for O(N^2)*
```python
class Solution(object):
    def minOperations(self, boxes):
        """
        :type boxes: str
        :rtype: List[int]
        """
        ans = []
        for i in range(len(boxes)):
            operation_cnt = 0
            for j in range(len(boxes)):
                if boxes[j] == "1":
                    operation_cnt += abs(j-i)
            ans.append(operation_cnt)
        return ans
```

*02  分成两次遍历：根据前一个盒子的操作数得到下一个盒子的操作数 O(N)*
```python
class Solution(object):
    def minOperations(self, boxes):
        """
        :type boxes: str
        :rtype: List[int]
        """
        left, right, operation = int(boxes[0]), 0, 0
        for i in range(1, len(boxes)):
            if boxes[i] == "1":
                right += 1
                operation += i
        ans = [operation]
        for i in range(1, len(boxes)):
            operation = operation + left - right
            if boxes[i] == "1":
                left += 1
                right -= 1
            ans.append(operation)
        return ans
```
