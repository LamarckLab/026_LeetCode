#### 3667  按绝对值排序数组
---


*01  随便写写*
```python
class Solution(object):
    def sortByAbsoluteValue(self, nums):
        """
        :type nums: List[int]
        :rtype: List[int]
        """
        for i in range(len(nums)-1):
            for j in range(i+1, len(nums)):
                if abs(nums[i]) > abs(nums[j]):
                    nums[i], nums[j] = nums[j], nums[i]
        return nums
```
