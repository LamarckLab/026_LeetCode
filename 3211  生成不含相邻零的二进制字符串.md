#### 3211  生成不含相邻零的二进制字符串
---


*01  随便写写 O(2^N)*
```python
class Solution(object):
    def validStrings(self, n):
        """
        :type n: int
        :rtype: List[str]
        """
        bin_list = []
        for i in range(0, 2**n):
            bin_str = bin(i)[2:].zfill(n)
            if "00" not in bin_str:
                bin_list.append(bin_str)
        return bin_list
```
