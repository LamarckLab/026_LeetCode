#### 3616  学生替换人数
---


*01  随便写写*
```python
class Solution(object):
    def totalReplacements(self, ranks):
        """
        :type ranks: List[int]
        :rtype: int
        """
        replace_cnt = 0
        smallest_stu = ranks[0]
        for digit in ranks:
            if digit < smallest_stu:
                smallest_stu = digit
                replace_cnt += 1
        return replace_cnt
```
