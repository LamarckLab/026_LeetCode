#### 0163  缺失的区间
---


*01 随便写写 o(n)*
```python
class Solution(object):
    def findMissingRanges(self, nums, lower, upper):
        """
        :type nums: List[int]
        :type lower: int
        :type upper: int
        :rtype: List[List[int]]
        """
        ans = []
        prev = lower - 1
        for num in nums:
            if num - prev >= 2:
                subRange = [prev+1,num-1]
                ans.append(subRange)
            prev = num
        if upper - prev >= 1:
            subRange = [prev+1,upper]
            ans.append(subRange)
        return ans
```
