#### 3146  两个字符串的排列差
---


*01  随便写写*
```python
class Solution(object):
    def findPermutationDifference(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: int
        """
        loc_diff = 0
        for letter in s:
            loc_diff += (abs(s.index(letter)-t.index(letter)))
        return loc_diff
```
