#### 0001 两数之和
---


*01  两个for暴力求解 O(N^2)*
```python
class Solution(object):
    def twoSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        """
        for i in range(len(nums)):
            for j in range(i+1,len(nums)):
                if nums[i] + nums[j] == target:
                    return [i,j]
```

*02  哈希表 O(N)*
```python
class Solution(object):
    def twoSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        """
        hashtable = dict()
        for index, num in enumerate(nums):
            left = target - num
            if left in hashtable:
                return [hashtable[left], index]
            hashtable[num] = index
```
