#### 3065  超过阈值的最少操作数 I
---


*01  随便写写*
```python
class Solution(object):
    def minOperations(self, nums, k):
        """
        :type nums: List[int]
        :type k: int
        :rtype: int
        """
        cnt = 0
        for digit in nums:
            if digit < k:
                cnt += 1
        return cnt
```
