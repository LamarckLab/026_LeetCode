#### 3110  字符串的分数
---


*01  随便写写*
```python
class Solution(object):
    def scoreOfString(self, s):
        """
        :type s: str
        :rtype: int
        """
        gap_sum = 0
        for index in range(len(s)-1):
            gap_sum += abs(ord(s[index])-ord(s[index+1]))
        return gap_sum
```
