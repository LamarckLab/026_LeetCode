#### 2974  最小数字游戏
---


*01  堆 O(NlogN)*
```python
class Solution(object):
    def numberGame(self, nums):
        """
        :type nums: List[int]
        :rtype: List[int]
        """
        num_stack = sorted(nums, reverse=True)
        arr = []
        while len(num_stack) > 0:
            alice = num_stack.pop()
            bob = num_stack.pop()
            arr.append(bob)
            arr.append(alice)
        return arr
```

*02  先排序后swap O(NlogN)*
```python
class Solution(object):
    def numberGame(self, nums):
        """
        :type nums: List[int]
        :rtype: List[int]
        """
        arr = sorted(nums)
        for i in range(len(arr)//2):
            arr[2*i], arr[2*i+1] = arr[2*i+1], arr[2*i]
        return arr
```
