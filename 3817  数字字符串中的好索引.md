#### 3817  数字字符串中的好索引
---


*01 随便写写*
```python
class Solution(object):
    def goodIndices(self, s):
        """
        :type s: str
        :rtype: List[int]
        """
        ans = []
        for i in range(len(s)):
            subStrLen = len(str(i))
            subStr = s[i+1-subStrLen:i+1]
            if int(subStr) == i:
                ans.append(i)
        return ans
```
