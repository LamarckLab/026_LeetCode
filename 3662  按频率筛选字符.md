#### 3662  按频率筛选字符
---


*01  随便写写*
```python
class Solution(object):
    def filterCharacters(self, s, k):
        """
        :type s: str
        :type k: int
        :rtype: str
        """
        output_s = ""
        for letter in s:
            if s.count(letter) < k:
                output_s += letter
        return output_s
```
