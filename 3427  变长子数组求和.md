#### 3427  变长子数组求和
---


*01  暴力切片 O(N^2)*
```python
class Solution(object):
    def subarraySum(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        output = 0
        for i in range(len(nums)):
            sublist = nums[max(0, i-nums[i]): i+1]
            output += sum(sublist)
        return output
```

*02  前缀和 O(N)*
```python
class Solution(object):
    def subarraySum(self, nums):
        """
        :type nums: List[int]
        :rtype: int·
        """
        prefix_sum = [0] * (len(nums)+1)
        output = 0
        for i in range(len(nums)):
            prefix_sum[i+1] = prefix_sum[i] + nums[i]
        for i in range(len(nums)):
            output += (prefix_sum[i+1] - prefix_sum[max(0, i-nums[i])])
        return output
```
