*01 动态规划 o(n^2)*
```python
class Solution(object):
    def getRow(self, rowIndex):
        """
        :type rowIndex: int
        :rtype: List[int]
        """
        triList = [[1],[1,1]]
        if rowIndex < len(triList):
            return triList[rowIndex]
        for i in range(2,rowIndex+1):
            subList = [1]*(i+1)
            for j in range(1,i):
                subList[j] = triList[i-1][j-1] + triList[i-1][j]
            triList.append(subList)
        return triList[rowIndex]
```
