#### 0206  反转链表
---


*01 迭代法 o(N)*
```python
# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution(object):
    def reverseList(self, head):
        """
        :type head: Optional[ListNode]
        :rtype: Optional[ListNode]
        """
        cur = head
        result = None
        while cur:
            temp = cur.next
            cur.next = result
            result = cur
            cur = temp
        return result
```
