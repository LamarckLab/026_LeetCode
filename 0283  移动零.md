#### 0283  移动零

*01 双指针 o(N)*
```python
class Solution(object):
    def moveZeroes(self, nums):
        """
        :type nums: List[int]
        :rtype: None Do not return anything, modify nums in-place instead.
        """
        nonZeroLoc = 0
        for i in range(len(nums)):
            if nums[i] != 0:
                nums[i], nums[nonZeroLoc] = nums[nonZeroLoc], nums[i]
                nonZeroLoc += 1
        return nums
```
