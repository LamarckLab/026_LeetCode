#### 1317  将整数转换为两个无零整数的和
---


*01  随便写写*
```python
class Solution(object):
    def getNoZeroIntegers(self, n):
        """
        :type n: int
        :rtype: List[int]
        """
        a = 1
        b = n - 1
        while True:
            if str(b).count("0") == 0 and str(a).count("0") == 0:
                return [a, b]
            else:
                a += 1
                b -= 1
```
