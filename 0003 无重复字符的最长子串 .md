#### 0003 无重复字符的最长子串 
---


*01 双指针+哈希表 O(N)*
```python
class Solution(object):
    def lengthOfLongestSubstring(self, s):
        """
        :type s: str
        :rtype: int
        """
        last = {}
        left = 0
        max_len = 0
        for right in range(len(s)):
            ch = s[right]
            if ch in last and last[ch] >= left:
                left = last[ch] +1
            last[ch] = right
            cur_len = right - left + 1
            max_len = max(cur_len, max_len)
        return max_len
```
