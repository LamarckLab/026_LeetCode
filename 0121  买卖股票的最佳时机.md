#### 0121  买卖股票的最佳时机
---


*01 贪心算法 o(n)*
```python
class Solution(object):
    def maxProfit(self, prices):
        """
        :type prices: List[int]
        :rtype: int
        """
        min_price = prices[0]
        max_profit = 0
        for price in prices:
            min_price = min(price,min_price)
            max_profit = max(max_profit,price-min_price)
        return max_profit
```
