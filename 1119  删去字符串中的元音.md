#### 1119  删去字符串中的元音
---


*01  随便写写*
```python
class Solution(object):
    def removeVowels(self, s):
        """
        :type s: str
        :rtype: str
        """
        vowel_list = ["a","e","i","o","u"]
        for letter in vowel_list:
            s = s.replace(letter,"")
        return s
```
