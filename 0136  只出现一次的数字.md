#### 0136  只出现一次的数字
---


*01 异或位运算 o(n)*
```python
class Solution(object):
    def singleNumber(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        ans = 0
        for num in nums:
            ans ^= num
        return ans
```
