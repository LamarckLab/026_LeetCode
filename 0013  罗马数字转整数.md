#### 0013  罗马数字转整数
---


*01 随便写写*
```python
class Solution(object):
    def romanToInt(self, s):
        """
        :type s: str
        :rtype: int
        """
        romaHash = {'I':1, 'V':5, 'X':10, 'L':50, 'C':100, 'D':500, 'M':1000}
        result = 0
        if len(s) == 1:
            result = romaHash[s]
        else:
            for i in range(len(s)-1):
                if romaHash[s[i]] >= romaHash[s[i+1]]:
                    result += romaHash[s[i]]
                else:
                    result -= romaHash[s[i]]
            result += romaHash[s[-1]]
        return result
```
